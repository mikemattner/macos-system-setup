# macos-system-setup
My system setup in most circumstances

## Prerequisites
* (Install Xcode Tools)[http://railsapps.github.io/xcode-command-line-tools.html]
* (Install Homebrew)[https://brew.sh/]
* (Install Oh My ZSH)[https://github.com/robbyrussell/oh-my-zsh]

## Installation
`source boostrap.sh`
